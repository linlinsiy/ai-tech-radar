# S30 简报 Markdown 表达与中文标题

**日期**：2026-07-19  
**范围**：内部 L5 简报生成 Prompt

## 调整内容

1. L5 Prompt 升级至 `v3.2`，要求每个主题标题使用中文；英文标题必须准确翻译，专有模型、框架、协议和产品名称可保留英文或中英并列。
2. 保持既有 Markdown 分类标题、主题标记和来源链接结构，兼容正文完整性校验。
3. 每个主题改为“事件与变化、技术要点与影响、内部参考”2至3个职责清晰的自然段。非技术资讯在第二段表达产品、市场、监管或生态影响，不虚构技术机制。
4. 不改变既有内容长度、L4 选题、来源链接、L5 分段生成和数据库结构。

## 验证

- 内部简报渲染与选择器测试覆盖 Markdown 主题标记、来源链接和固定组装结构。
