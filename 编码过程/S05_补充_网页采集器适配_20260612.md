#### 量子位（qbitai.com）网页采集器实现

**产出文件**：[tech-research/aws/app/crawler/web_crawler.py](../tech-research/aws/app/crawler/web_crawler.py)

**HTML 结构分析**：
| 区域 | 选择器 | 说明 |
|---|---|---|
| 轮播/焦点文章 | div.swiper-slide a[href] | 头条区域，含 h3 标题和图片 |
| 常规文章列表 | h4 a[href] + 相邻 span.time | 正文列表，含标题和相对时间 |
| 文章详情-标题 | h1 | 文章页标题 |
| 文章详情-作者 | span.author a | 作者链接 |
| 文章详情-时间 | span.time | 发布时间（绝对或相对格式） |
| 文章详情-正文 | div.content | 主体内容区域 |

**实现功能**：
1. **_parse_qbitai_list()** — 解析首页文章列表
   - 三种选择器覆盖焦点文章、常规文章、兜底嗅探
   - seen_urls 集合去重
   - 自动分页（/page/2、/page/3），最多 3 页

2. **_parse_qbitai_relative_time()** — 相对时间解析
   - 支持绝对日期：YYYY-MM-DD / YYYY/MM/DD
   - 支持相对时间：N小时前 / N分钟前 / N天前 / N周前 / 昨天

3. **_parse_qbitai_detail()** — 文章详情页解析
   - 提取标题、作者、发布时间、正文
   - 自动清理 script/style/.info/.tags_s 等无关元素

4. **WebCrawler._fetch_qbitai()** — 采集编排
   - 列表解析 → 列表页已有时间直接使用
   - 列表页无时间则回源抓取详情页
   - 组装 RawArticle，max_articles 上限 100 篇

**代码量**：骨架 69 行 → 实现 290 行
